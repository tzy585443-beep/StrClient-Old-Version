# StrClient (Old Version)

A Minecraft client built for **Fabric 1.21.1**.

> [!WARNING]
> This repository contains an **old version** of StrClient and is no longer actively maintained.

## About

StrClient is a personal Minecraft client project created for learning and experimenting with Minecraft mod development on the Fabric platform.

This repository is intended to demonstrate topics such as:

* Event system
* Module system
* GUI / HUD implementation
* Rendering
* Configuration management
* General client architecture

## Educational Purpose

This project is provided **for educational and research purposes only**.

It is **not designed or maintained to bypass any anti-cheat system**, evade server security, or violate the rules of any Minecraft server.
